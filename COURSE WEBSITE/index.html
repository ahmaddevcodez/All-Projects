What Is HTML?

HTML stands for Hyper Text Markup Language. It is a standard markup language for web page creation.It is used to make website. (It Is Not a Programming Language)

HTML consist of element that describe headings, text, tables, lists, photos, etc. Retrieve online information via hypertext links, at the click of a button.

In HTML, each element is made up of three part

  1. Opening tag
  2. Closing tag
  3. The actual element
